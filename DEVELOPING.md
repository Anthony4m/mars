# Developing Mars

- Go version: 1.21

## Build
```
go build ./...
```

## Run REPL
```
go run ./cmd/mars repl
```

## Run a file
```
go run ./cmd/mars run examples/two_sum_working_final.mars
```

## Run tests
```
go test ./...
```
