# Start here

- Install Go 1.22+
- Clone the repo, then build the CLI:

```
go build ./cmd/mars
```

- Hello world:

```
func main() {
  println("Hello, Mars!")
}
```

Run it:

```
go run ./cmd/mars run examples/hello.mars
```

- Next steps:
  - Tour built-ins in `evaluator/builtins.go`
  - Read `docs/tutorial.md`
  - Browse `examples/`
  - Quick-reference: `docs/quick-reference.md`
